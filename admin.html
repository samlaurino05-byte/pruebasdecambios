Panel administrador v3

Versión corregida para tu Excel actual: el horario base se mantiene siempre, excepto cuando elegís explícitamente cambio de horario. Las excepciones con fecha vencen solas y la materia vuelve automáticamente al horario/aula original.

1) Identificar materia

Usá exactamente la combinación materia + comisión + día + horario. El horario se usa solo para encontrar la fila correcta, no para cambiarla.

Busca en columnas reales: materia, comision, dia, horario_inicio, horario_fin.

2) Cambio definitivo

Usalo solo para cambios permanentes. Por defecto NO toca el horario.

Si elegís “solo aula/edificio”, el script no toca horario_inicio ni horario_fin.

3) Crear excepción

Para cambios temporales o indeterminados. La base queda intacta; la excepción solo pisa el dato mientras está vigente.

Cómo revierte automáticamente:
• Solo un día: aplica solo en esa fecha.
• Duración específica: aplica desde fecha inicio hasta fecha fin inclusive.
• Hasta nuevo aviso: aplica desde la fecha inicio mientras esté activo en “SI”.
Regla central: si la excepción no está vigente hoy, la pantalla usa el horario original del Excel.

4) Agregar materia nueva

Inserta una fila nueva en Materias con las columnas que usa tu pantalla pública.

5) Eliminar materia base

Elimina la fila base buscada. Las excepciones asociadas no se muestran si la base ya no existe.

Se elimina por la misma combinación: materia + comisión + día + horario.
